# Fluxo do Sistema — Mão na Massa

## 1. Cadastro de Serviço

O prestador de serviços cadastra um serviço na plataforma, informando detalhes como descrição, categoria, preço e disponibilidade.

⬇️

## 2. Criação da Solicitação

O cliente visualiza os serviços disponíveis e cria uma solicitação para o serviço desejado.

⬇️

## 3. Criação do Orçamento

O prestador analisa a solicitação recebida e envia um orçamento referente àquela solicitação específica.

⬇️

## 4. Decisão do Cliente

O cliente analisa o orçamento recebido e possui duas opções:

* ❌ **Recusar orçamento**
  → Processo encerrado.

* ✅ **Aceitar orçamento**
  → Cliente pode prosseguir para o agendamento.

⬇️

## 5. Agendamento

Após aceitar o orçamento, o cliente realiza o agendamento do serviço.

⬇️

## 6. Decisão do Prestador

O prestador analisa o agendamento solicitado e possui duas opções:

* ❌ **Recusar agendamento**
  → Processo encerrado.

* ✅ **Aceitar agendamento**
  → Solicitação confirmada.

⬇️

## 7. Execução do Serviço

O prestador realiza o serviço conforme o agendamento confirmado.

⬇️

## 8. Avaliação

Após a conclusão do serviço, o cliente realiza uma avaliação do prestador, podendo atribuir nota e comentário.
